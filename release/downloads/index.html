Version 19.0 (06.05.2025)

Diese Version baut auf dem erfolgreichen Ansatz von v18 auf und bietet eine verbesserte Benutzeroberfläche mit robuster VMDK-Erkennung.

Neue Funktionen:
  • Vollständig überarbeitete Weboberfläche mit Bootstrap 5
  • Responsives Design für Desktop und mobile Geräte
  • Verbesserte Darstellung von VMware Tools Status
  • Erweiterte Snapshot-Altersanalyse mit Farbkodierung
  • Zuverlässigere Erkennung verwaister VMDK-Dateien

Installationshinweise
Systemvoraussetzungen
  • Python 3.8 oder höher
  • Internetverbindung für die Installation der Abhängigkeiten
  • Zugriff auf einen VMware vCenter-Server (nicht erforderlich für Demo-Modus)
Installation unter Windows
  1. Entpacken Sie das ZIP-Archiv
  2. Führen Sie setup.bat aus
  3. Starten Sie die Anwendung mit run.bat
Installation unter Linux
  1. Entpacken Sie das ZIP-Archiv
  2. Führen Sie chmod +x setup.sh run.sh aus
  3. Führen Sie ./setup.sh aus
  4. Starten Sie die Anwendung mit ./run.sh
