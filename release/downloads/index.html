Systemanforderungen
Mindestanforderungen
  • Python 3.8 oder höher
  • 2 GB RAM
  • 500 MB freier Festplattenspeicher
  • Netzwerkverbindung zum vCenter Server
Unterstützte Betriebssysteme
  • Windows 10/11
  • Windows Server 2019/2022
  • OpenSUSE Tumbleweed
  • Andere Linux-Distributionen (mit Python 3.8+)
Installationsanleitung
Windows-Installation
  1. Laden Sie die ZIP-Datei herunter und entpacken Sie sie
  2. Führen Sie setup.bat aus, um alle Abhängigkeiten zu installieren
  3. Starten Sie die Anwendung mit run.bat
  4. Öffnen Sie in Ihrem Browser: http://localhost:5000
Linux-Installation
  1. Laden Sie die ZIP-Datei herunter und entpacken Sie sie
  2. Führen Sie chmod +x setup.sh aus, um die Skripte ausführbar zu machen
  3. Führen Sie ./setup.sh aus, um alle Abhängigkeiten zu installieren
  4. Starten Sie die Anwendung mit python run.py
  5. Öffnen Sie in Ihrem Browser: http://localhost:5000
