====================================================
VMware vSphere Reporter Version 19.0 - Änderungsprotokoll
====================================================

Version 19.0 (06.05.2025)
-------------------------

Diese Version baut auf dem erfolgreichen Ansatz von v18 auf und bietet eine verbesserte Benutzeroberfläche mit robuster VMDK-Erkennung.

NEUE FUNKTIONEN:
- Vollständig überarbeitete Weboberfläche mit Bootstrap 5
- Responsives Design für Desktop und mobile Geräte
- Verbesserte Darstellung von VMware Tools Status
- Erweiterte Snapshot-Altersanalyse mit Farbkodierung
- Zuverlässigere Erkennung verwaister VMDK-Dateien
- Umfassende Rohdaten-Anzeige für Diagnose und Fehlersuche
- Detaillierte Fehlerprotokolle für eine bessere Fehlerbehebung

VERBESSERUNGEN:
- Optimierte VMDK-Erkennung basierend auf dem erfolgreichen Ansatz von v18
- Robustere Fehlerbehandlung für stabilere Datenerfassung
- Bessere Filterung von Template- und Snapshot-VMDKs
- Verbesserte Benutzerführung und Informationsanzeige
- Professionelles Bechtle-Farbschema in der gesamten Anwendung

TECHNISCHE ÄNDERUNGEN:
- Überarbeitete Client-Struktur für eine bessere Wartbarkeit
- Detailliertere Protokollierung für Fehlerbehebung
- Verbesserte Statusanzeige für Datensammlungen
- Angepasste Demo-Daten für realistischere Anzeigen

INSTALLATIONSÄNDERUNGEN:
- Verbesserte Setup-Skripte für Windows und Linux
- Unterstützung für virtuelle Python-Umgebungen
- Automatische Port-Auswahl für den Webserver
- Verbesserte Fehlerbehandlung bei der Installation

BEKANNTE PROBLEME:
- Berichterstellung im PDF- und DOCX-Format noch nicht implementiert
- Anmeldung mit Benutzeranmeldeinformationen kann in einigen Netzwerkkonfigurationen fehlschlagen

HINWEISE:
- Unterstützt nun Python 3.8 und höher (vorherige Version: Python 3.9+)
- Benötigt pip und die Möglichkeit zur Installation von Paketen
- Detaillierte Installationsanweisungen finden Sie in der README.md-Datei
