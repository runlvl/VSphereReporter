Bechtle vSphere Reporter v0.1

Willkommen beim Download des Bechtle vSphere Reporters in der Version 0.1. Diese webbasierte Anwendung ermöglicht umfassende Berichterstellung für VMware vSphere-Umgebungen.

Hauptmerkmale:

  • Vollständige webbasierte Oberfläche für plattformunabhängigen Zugriff
  • Berichterstellung für VMware Tools-Versionen
  • Erkennung und Altersanalyse von Snapshots
  • Zuverlässige Erkennung verwaister VMDK-Dateien
  • Export in HTML-, DOCX- und PDF-Format
  • Einfache Installation durch Smart-Starter-Skripte

Klicken Sie auf den folgenden Button, um das Tool herunterzuladen:

Bechtle vSphere Reporter v0.1 herunterladen

Hinweis: Für Linux wird empfohlen, nach dem Entpacken das Skript start.sh zu verwenden. Windows-Nutzer können die Anwendung bequem mit start.bat starten.
