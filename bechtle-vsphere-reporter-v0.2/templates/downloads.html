Systemanforderungen
Mindestanforderungen
  • Python 3.8 oder höher
  • 2 GB RAM
  • 500 MB freier Festplattenspeicher
  • Netzwerkverbindung zum vCenter Server
Unterstützte Betriebssysteme
  • Windows 10/11
  • Windows Server 2019/2022
  • OpenSUSE Tumbleweed
  • Andere Linux-Distributionen (mit Python 3.8+)
Installationsanleitung
Windows-Installation
  1. Laden Sie die ZIP-Datei herunter und entpacken Sie sie
  2. Führen Sie start.bat aus, um die Anwendung zu starten
  3. Öffnen Sie in Ihrem Browser: http://localhost:5000 oder die in der Konsole angezeigte URL
Linux-Installation
  1. Laden Sie die ZIP-Datei herunter und entpacken Sie sie
  2. Führen Sie chmod +x start.sh aus, um das Startskript ausführbar zu machen
  3. Führen Sie ./start.sh aus, um die Anwendung zu starten
  4. Öffnen Sie in Ihrem Browser: http://localhost:5000 oder die in der Konsole angezeigte URL
