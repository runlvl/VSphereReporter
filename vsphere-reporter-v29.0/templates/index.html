Hauptfunktionen

Entdecken Sie die leistungsstarken Features des VMware vSphere Reporters

VMware Tools Status

Identifizieren Sie veraltete oder nicht installierte VMware Tools-Versionen auf all Ihren virtuellen Maschinen.

Snapshot-Analyse

Erkennen Sie alte Snapshots, die potenziell Speicherplatz verschwenden und die Performance beeinträchtigen können.

Verwaiste VMDKs

Finden Sie verwaiste VMDK-Dateien, die keiner VM zugeordnet sind und wertvollen Speicherplatz belegen.

Infrastruktur-Topologie

Visualisieren Sie Ihre gesamte vSphere-Infrastruktur mit interaktiven Diagrammen.

Multi-Format-Exporte

Exportieren Sie Berichte in verschiedenen Formaten: HTML, PDF und DOCX.

Browser-Zugriff

Greifen Sie von überall aus über einen Webbrowser auf den Reporter zu, ohne lokale Installation.

Vorteile der Web-Version

Was ist neu in Version 29.0

Plattformunabhängigkeit

Mit der neuen webbasierten Architektur können Sie den VMware vSphere Reporter von jedem Gerät mit einem modernen Browser aus verwenden - egal ob Windows, Linux, macOS oder sogar Mobilgeräte.

  • Keine lokale Installation erforderlich
  • Einfache Bereitstellung in Unternehmensnetzwerken
  • Zentralisierter Zugriff für mehrere Benutzer
Verbesserte Benutzeroberfläche

Das neue responsive Design bietet eine moderne, benutzerfreundliche Oberfläche mit verbesserter Lesbarkeit und einfacherer Navigation.

  • Interaktive Visualisierungen
  • Optimiert für Desktop- und Mobilgeräte
  • Intuitiver Arbeitsablauf für Berichtserstellung
Verbesserter VMDK-Sammler

Der neue VM-zentrische Ansatz zur Erkennung von verwaisten VMDK-Dateien liefert genauere Ergebnisse und reduziert falsch positive Meldungen.

  • Verbesserte Erkennung von tatsächlich verwaisten VMDKs
  • Intelligente Klassifizierung von VMDK-Typen
  • Detaillierte Informationen zu jeder potenziell verwaisten VMDK
Robuste Fehlerbehandlung

Das neue Fehlerbehandlungssystem bietet klare, benutzerfreundliche Meldungen und erhöht die Stabilität der Anwendung erheblich.

  • Detaillierte Fehlerprotokolle für Administratoren
  • Benutzerfreundliche Fehlermeldungen
  • Automatische Wiederherstellung nach bestimmten Fehlern
