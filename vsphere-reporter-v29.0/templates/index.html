Hauptfunktionen

VMware Tools Analyse

Erkennen Sie veraltete VMware Tools Versionen und aktualisieren Sie Ihre VM-Umgebung.

Snapshot-Management

Identifizieren Sie alte Snapshots, die Speicherplatz und Performance beeinträchtigen können.

VMDK-Analyse

Finden Sie verwaiste VMDK-Dateien und gewinnen Sie wertvollen Speicherplatz zurück.

Infrastruktur-Topologie

Visualisieren Sie Ihre vSphere-Umgebung mit interaktiven Netzwerk-Diagrammen.

Multi-Format Export

Exportieren Sie Berichte in HTML, PDF oder DOCX zur einfachen Verteilung.

Sicherheit

Sichere Verbindung zum vCenter ohne Speicherung von Anmeldedaten.

Schnellstart-Anleitung

  1. Verbindung zum vCenter

    Geben Sie Ihre vCenter-Serveradresse, Benutzernamen und Passwort ein.

  2. Berichtsoptionen auswählen

    Wählen Sie die gewünschten Berichte und Exportformate aus.

  3. Bericht generieren

    Starten Sie die Analyse und warten Sie auf den generierten Bericht.

  4. Ergebnisse herunterladen

    Laden Sie den fertigen Bericht herunter oder teilen Sie ihn direkt im Browser.
