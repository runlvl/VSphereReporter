Hinweise zur Verbindung
Benötigte Berechtigungen

Der Benutzer benötigt mindestens Leserechte auf die vSphere-Umgebung, um Berichte zu generieren. Für vollständige Funktionalität empfehlen wir die folgenden Berechtigungen:

  • Leserechte für virtuelle Maschinen
  • Leserechte für ESXi-Hosts
  • Leserechte für Datastores
  • Leserechte für Netzwerke
Verbindungsprobleme?
  • Stellen Sie sicher, dass der vCenter Server erreichbar ist
  • Überprüfen Sie Benutzername und Passwort
  • Prüfen Sie, ob der Browser Zugriff auf Port 443 hat
  • Bei selbstsignierten Zertifikaten die entsprechende Option aktivieren
