Hinweise
Berichtsdetails

Der Bericht sammelt Informationen zu den folgenden Bereichen, abhängig von Ihrer Auswahl:

  • VMware Tools: Prüft VMware Tools-Status und -Versionen auf allen VMs
  • Snapshots: Identifiziert alte Snapshots, sortiert nach Alter
  • Verwaiste VMDKs: Findet nicht zugeordnete VMDK-Dateien
  • Infrastruktur: ESXi-Hosts, Datastore, Netzwerke und mehr
Berichtsformat

{% if options.get('export_format') == 'html' %} Der HTML-Bericht bietet interaktive Funktionen wie Sortierung, Filterung und detaillierte Ansichten. {% elif options.get('export_format') == 'pdf' %} Das PDF-Format ist optimal für die Dokumentation und das Ausdrucken des Berichts. {% elif options.get('export_format') == 'docx' %} Das DOCX-Format erlaubt die nachträgliche Bearbeitung des Berichts in Microsoft Word. {% else %} Der Bericht wird in einem leicht lesbaren Format erstellt. {% endif %}

Tipp: Große vSphere-Umgebungen können zu längeren Verarbeitungszeiten führen, besonders bei der Analyse verwaister VMDKs. Bitte haben Sie Geduld während der Berichtsgenerierung.
