VMware Tools Status

Übersicht über den VMware Tools-Status aller virtuellen Maschinen, sortiert nach Version (älteste zuerst).

Informationen

Der VMware Tools-Status zeigt an, ob die virtuellen Maschinen aktuelle VMware Tools installiert haben.

  • Aktuell - Die neueste Version der VMware Tools ist installiert
  • Veraltet - Es ist eine ältere Version der VMware Tools installiert
  • Nicht installiert - VMware Tools sind nicht installiert
  • Nicht ausgeführt - VMware Tools sind installiert, werden aber nicht ausgeführt

Für eine optimale Leistung und Funktionalität sollten alle VMs aktuelle VMware Tools verwenden.
